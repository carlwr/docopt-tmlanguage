## QA

```sh
npm run qa     # lint (check) + grammar tests
```

## Release

```sh
# bump version: runs qa, edits package.json, commits, tags, pushes
npm run version:patch   # 0.1.0 -> 0.1.1
npm run version:minor   # 0.1.0 -> 0.2.0
npm run version:major   # 0.1.0 -> 1.0.0

# publish to npm (prepublishOnly runs qa first)
npm publish
```
